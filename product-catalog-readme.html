Product Catalog README

The goal of the Product Catalog Webpage is to demonstrate taking raw data from a Excel format and transforming it into a format that can be displayed to an end user with intuitive information and search functionality.

  • The raw data can be found in “Product Catalog Database.csv” and includes:
    • Vendor Part Number
    • UPC
    • Product Name
    • Count
    • Weight
    • Category
    • Sub-Category
    • In-House Part Number
    • Material
    • Organic
    • Size
    • Inventory
    • Vendor
    • Price
    • ETA
  • This raw data is then fed into “Product Catalog Database – MACRO”
    • The Vendor column is transformed into the file path of their relative brand logos.
    • A Image column was added with the applicable category and/or sub-category file path images.
    • The next step utilizes a VBA macro to transform into a JSON format data table to feed into the webpage code.
    • This JSON data table is then fed into the HTML webpage code.
  • Product Catalog Webpage
    • The entire product catalog is displayed for searching.
    • Search and be done using the search bar.
    • This search bar searches fields including:
      • Vendor Part Number
      • UPC
      • Product Name
      • Count
      • Weight
      • Category
      • Sub-Category
      • In-House Part Number
      • Material
      • Organic
      • Size
      • Inventory
      • Vendor
      • Price
      • ETA
    • A text color condition is coded to show “True” and green if the product is organic.
    • Brand logos and product images are displayed in consistent pixel formats.
    • A text color condition is coded to show “True” and green if the product is organic.

    • Add to cart function
    • Filtering and searching by each data field
    • Filtering and sorting by price, ETA and available
