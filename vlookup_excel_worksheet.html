These three worksheets provide an example and excerise demonstraing Excel VLOOKUP and Advanced Filters.

VLOOKUP Sheet: The purpose of the "VLOOKUP" sheet is to use VLOOKUP to reference the values in cells L3 through L7 and return their corresponding data from columns A through J. Additionally, the returned results are formatted to simulate a shopping cart experience, providing a clear and intuitive view of selected items.

  • =VLOOKUP(L3, A:J, 2, FALSE)
  • =VLOOKUP(L3, A:J, 6, FALSE)
  • =VLOOKUP(L3, A:J, 8, FALSE)
  • =VLOOKUP(L3, A:J, 9, FALSE)
  • =SUM(R3:R9)
  • =SUM(R10:R11)

Advanced Filters: The purpose of the "Advanced Filters" sheet is to filter values of D2 "Food Product" and E2 "Dairy" from the Catalog Database using Excel Advanced Filters and return all rows of relative values.

  • Filter with values from D2 and E2 in A5:J102
  • Filter the list, in-place
  • List Range: $A$5:$J$102
  • Critera Range: $A$1:$J$2

SEARCH: The "Search" sheet utilizes the FILTER function with ISNUMBER and SEARCH. A user can input text into the Product Name search box (M2) and the functionse searches the name field from the catalog base and returns their correlating values into L6:U6

  • =FILTER(A6:J102, ISNUMBER(SEARCH(M2, A6:A102)), "ERROR")
